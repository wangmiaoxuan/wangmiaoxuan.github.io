Moment 日期和時間示範

Moment github

範例程式碼

        

            <div class="border p-4 rounded" style="font-size: larger;">
                <p>當前時間：<span id="current-time"></span></p>
                <p>格式化時間：<span id="formatted-time"></span></p>
                <p>從現在起的10天後：<span id="future-date"></span></p>
            </div>

            <script>
                // 當前時間
                const currentTime = moment();
                document.getElementById('current-time').textContent = currentTime.format('YYYY-MM-DD HH:mm:ss');
                // 格式化時間
                document.getElementById('formatted-time').textContent = currentTime.format('dddd, MMMM Do YYYY');
                // 未來日期
                const futureDate = currentTime.add(10, 'days');
                document.getElementById('future-date').textContent = futureDate.format('YYYY-MM-DD');
            </script>
