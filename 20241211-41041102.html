API 範例介紹



股票資訊 API

股票資訊 API 提供台灣股票每日交易數據，包括股票代碼、成交量、收盤價等資訊，用於金融分析或即時股市追蹤。

查看股票資訊
電影院資訊 API

電影院資訊 API 提供全台電影院位置及廳數。

查看電影院資訊
空氣品質 API

空氣品質 API 提供全台各地的即時空氣品質數據（AQI），包括 PM2.5、PM10、主要污染物等，適合用於環境監測應用。

查看空氣品質
記事本 API

記事本 API 支援備忘錄功能，可用於記事工作、個人之重要事件，並透過GOOGLE帳號註冊，以標示並記錄完成或未完成事件，隨時隨地可編輯及刪除。

查看記事本
